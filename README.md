# Grupo-Burne-Zanini-V2


Este repositorio funciona como continuación del repositorio Grupo-Burne-Zanini. Debido a que 
el límite de tamaño para un push es de 100 Mb, y que en el historial del viejo repositorio existen archivos muy pesados (mediciones), el repositorio quedó estancado.
Este nuevo repositorio continúa desde el último estado commiteado.
